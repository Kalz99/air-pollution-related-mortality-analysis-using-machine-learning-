## Overview
This project explores the impact of air pollution on human mortality using machine learning techniques. By analyzing environmental indicators such as PM2.5, PM10, and NO₂, the project aims to identify patterns, predict mortality rates, and uncover hidden relationships within the data.

The goal is not only prediction, but also **interpretability and insight generation** to better understand how pollution affects public health.

---

## Objectives
- Analyze the relationship between air pollution and mortality rates  
- Build predictive models for estimating death rates  
- Apply clustering techniques to group regions with similar pollution patterns  
- Generate meaningful insights through data visualization  

---

## Machine Learning Approach

### Supervised Learning
- Regression models used to predict mortality rates
- Models experimented:
  - Linear Regression
  - Random Forest Regressor
  - KNN
  - XGBoost 

### Unsupervised Learning
- Clustering used to identify patterns across regions
- Algorithms:
  - K-Means Clustering
  - Random Forest
  - KNN
  - Hierarchical Clustering (optional)

---

## Dataset
- Environmental and health-related datasets (e.g., WHO or similar sources)
- Features include:
  - PM2.5 levels and other air pollutants
  - Hospital Beds 
  - Mortality rate (target variable)

---


## Key Insights
- Strong correlation observed between fine particulate matter (PM2.5) and mortality rates  
- Certain regions cluster together based on pollution exposure and health outcomes  
- Non-linear models outperform simple regression in capturing complex relationships
