Air Pollution & Mortality Analysis using Machine Learning

This project explores the relationship between air pollution levels (e.g., PM2.5, PM10, NO₂) and mortality rates using machine learning techniques. 
It combines data preprocessing, exploratory data analysis (EDA), and both supervised and unsupervised learning approaches to uncover patterns and insights.

Key features:

Data cleaning and feature engineering on environmental datasets
Exploratory analysis to identify trends and correlations
Supervised models to predict mortality rates
Unsupervised learning (clustering) 
Visualization of insights for better interpretability
